Solid from 10.000 ft

Solid is a specification that lets people store their data securely in decentralized data stores called Pods. Pods are like secure personal web servers for data. When data is stored in someone's Pod, they control which people and applications can access it.

From solidproject.org

  • Started by Tim Berners-Lee
  • Origin: Social Linked Data - SoLiD (not used anymore)

Pros & Cons - Developers

  • + Reusable and extensible data
  • + GDPR-compliant
  • + Good combo w/headless apps
  • - Complicated
  • - Specifications under development
  • - No significant network effect (yet)

Pros & Cons - Users

  • + No lock-in
  • + Reusable data
  • + Allows full control of data
  • - Few apps
  • - Challenging UX
